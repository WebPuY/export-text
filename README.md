# honey-jan-export
find words from word docs and export a excel

### Installation:

```
$ npm install -g honey-jan-export
$ cd your_project
$ honey-jan-export
```

### Other params:
* port: 自定义端口号， 默认是3000
* dir: 自定义目录， 默认在当前执行命令目录下
* key: 变量名, 以|隔开

eg.
```javascript
honey-jan-export port=3333 words=test1|test2|test3
```
